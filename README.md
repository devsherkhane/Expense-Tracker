#  Expense Tracker (Go + Vue + MySQL)

A full-stack Expense Tracker web application built using Go (Gin Framework), Vue.js, and MySQL.  
This project allows users to manage daily expenses by adding, viewing, and deleting records.

---

##  Features

- Add new expenses
- View all expenses
- Delete expenses
- Categorize expenses
- Store data in MySQL
- RESTful API using Go (Gin)
- Frontend built with Vue 3

---

##  Tech Stack

### Backend
- Go (Golang)
- Gin Framework
- GORM
- MySQL Driver

### Frontend
- Vue.js (Vue 3)
- Vite
- Axios
