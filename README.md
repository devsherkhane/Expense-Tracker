#  Expense Tracker 

A lightweight, high-performance Expense Tracker designed to manage daily financial records. This project serves as a robust example of a modern full-stack application using a Go backend and a Vue.js frontend, with persistent storage managed by MySQL.


---

##  Features

* Expense Management: Effortlessly add, view, and delete daily expense records.

* Categorization: Organize spending into specific categories for better financial tracking.

* RESTful Architecture: A clean API built with the Gin framework to handle data transactions.

* Reactive UI: A fast and responsive frontend built with Vue 3 and Vite for a seamless user experience.

* Database Persistence: Secure and structured data storage using MySQL.

---

##  Tech Stack

### Backend (The Engine)

* Go (Golang): Chosen for its concurrency and speed.

* Gin Framework: Used for routing and handling HTTP requests efficiently.

* GORM: An ORM for Go to manage database interactions and migrations.

* MySQL: Reliable relational database for storing expense history.

---

### Frontend (The Interface)

* Vue.js 3: Leveraging the Composition API for reactive state management.

* Vite: A modern build tool providing an extremely fast development environment.

* Axios: Used for making asynchronous API calls to the Go backend.
